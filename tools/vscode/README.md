# Peel Language Support

This is the VSCode extension providing language support for the Peel programming language.

## Features

- **Syntax Highlighting**: Supports keywords, types, strings, and operators in Peel.
- **Language Server Protocol (LSP)**: Automatic diagnostics for parse errors and type errors via red squiggles.
- **Intellisense**: Contextual autocomplete for standard language keywords.
- **Hover Support**: Easy symbol tooltip checking.

## Usage

Currently, this extension interacts with the `peel lsp` command. You must either have the `peel` application built in your original workspace, or update the source paths manually.
